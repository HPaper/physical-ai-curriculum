# /study-start

새 세션(또는 컨텍스트 리셋 후 새 채팅, 또는 다른 기기에서 붙은 세션)에서 학습을 이어가기 위한 리추얼.

수행할 것:

1. `STUDY_LOG.md`를 읽고 다음을 사용자에게 간결히 요약한다:
   - **Currently studying**: 어떤 강의를 어디까지 하고 있었는지.
   - **▶ NEXT**: 다음 마이크로 스텝으로 명시된 내용.
   - **Last updated**: 언제·어느 기기에서 마지막으로 갱신됐는지.
2. `Currently studying`의 강의 파일(경로 유추: `partNN-*/lecNN_*.md` 또는 README 표에서 확인)을 열어 **문서 하단의 `<!-- STUDY: -->` 개인 노트**가 있으면 함께 요약한다.
3. **오늘의 마이크로 스텝 1–3개**를 구체적으로 제안한다. 각 스텝은:
   - 결과물이 뚜렷해야 함(예: "그림 3개 추가", "흔한 오해 섹션 신설", "worked example 손계산 검증").
   - 30–60분 안에 리뷰 가능한 diff로 끝날 크기.
4. 사용자가 스텝을 선택할 때까지 **편집을 시작하지 않는다**. 필요한 추가 정보만 사용자에게 확인.

CLAUDE.md의 강의 파일 규약(핵심 수식 3단·worked example·그림 ≥5·검증 코드·이전/다음 네비)을 항상 준수.
